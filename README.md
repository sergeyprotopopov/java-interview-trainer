## Как продолжать обучение

1. Иди по темам сверху вниз, в каждой — формат **В: вопрос / О: ответ**.
2. Сначала отвечай сам, потом сверяйся с ответом.
3. Особое внимание — файлу [weak-spots.md](weak-spots.md): там темы, где на реальном собесе были ошибки.
4. Бери приоритеты из [learning-backlog.md](learning-backlog.md): темы из чек-листа собеседований, отсортированные по критичности (6 → 1), с пометкой пробелов (⛔ gap).
5. Добавляй новые вопросы в конец соответствующего файла по мере подготовки.
